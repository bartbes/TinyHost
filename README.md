**TinyHost** is simple bash script that uses a few small, basic utilities to 'upload' files to [TinyUrl][].

###Needed utilities###

    awk
    base64
    curl
    file
    perl

###Usage###

    tinyhost.sh <filename>

For example:

    tinyhost.sh myfile.html
